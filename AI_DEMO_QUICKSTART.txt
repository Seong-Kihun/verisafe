🚀 시연 방법 선택
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌─────────────────────────────────────────────────────────────────────┐
│ 방법 1: 전체 자동 시연 (20분) - 가장 간단!                          │
└─────────────────────────────────────────────────────────────────────┘

  1. DEMO_FULL.bat 더블클릭
     → 모델 학습 + 테스트 자동 실행

  2. DEMO_3_API.bat 더블클릭 (새 터미널)
     → API 서버 시작

  3. DEMO_4_PREDICT.bat 더블클릭 (또 다른 새 터미널)
     → API 실시간 예측 테스트


┌─────────────────────────────────────────────────────────────────────┐
│ 방법 2: 단계별 시연 (25분) - 설명하기 좋음                          │
└─────────────────────────────────────────────────────────────────────┘

  1단계: DEMO_1_TRAIN.bat      (10-15분)
         → 모델 학습

  2단계: DEMO_2_TEST.bat       (1-2분)
         → 모델 테스트

  3단계: DEMO_3_API.bat        (새 터미널, 계속 실행)
         → API 서버 시작

  4단계: DEMO_4_PREDICT.bat    (또 다른 새 터미널)
         → API 테스트


┌─────────────────────────────────────────────────────────────────────┐
│ 방법 3: 빠른 시연 (3분) - 이미 학습된 경우                          │
└─────────────────────────────────────────────────────────────────────┘

  1. DEMO_2_TEST.bat
     → 테스트만 실행하여 결과 보여주기
