# VeriSafe 개발 환경 설정 체크리스트

## ✅ 필수 환경 요구사항

### 1. 운영체제
- [x] Windows 10

### 2. 백엔드
- [ ] Python 3.11 이상 설치
  ```bash
  python --version
  ```
- [ ] PostgreSQL 14+ 설치
  ```bash
  psql --version
  ```
- [ ] PostGIS 설치 (PostgreSQL 확장)
- [ ] Redis 설치
  ```bash
  redis-cli --version
  ```

### 3. 프론트엔드
- [ ] Node.js 18+ 설치
  ```bash
  node --version
  npm --version
  ```
- [ ] Expo CLI 설치
  ```bash
  npm install -g expo-cli
  ```

### 4. 개발 도구
- [ ] Git 설치
- [ ] VS Code 또는 Cursor
- [ ] pgAdmin (선택사항, DB 관리용)

## 🔑 필요한 API 키

### ❌ Google Maps API 불필요!

**OpenStreetMap 사용**:
- ✅ **완전 무료**: API 키 불필요
- ✅ **제한 없음**: 타일 다운로드 무제한
- ✅ **오픈소스**: 자유롭게 사용 가능

**사용 방법**:
- React Native Maps에서 OpenStreetMap 타일 서버 URL 사용
- 예: `https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png`

**참고**: 
- OpenStreetMap은 Open Data Commons Open Database License (ODbL) 라이선스
- 상업적 이용 가능 (다만 OSM 데이터를 사용한다면 공개 의무)
