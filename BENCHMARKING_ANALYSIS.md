# VeriSafe 상용 네비게이션 벤치마킹 분석

**작성일**: 2025-11-06
**목적**: 상용 네비게이션 핵심 기능을 VeriSafe의 "안전 중심" 목적에 맞게 벤치마킹

---

## 🎯 VeriSafe 핵심 차별점 (기획안 기반)

| 항목 | Google Maps | UN OCHA | **VeriSafe** |
|------|-------------|---------|--------------|
| **목적** | 속도/효율 | 현황 참고 | **생존** 🎯 |
| **정보 실시간성** | 교통정보만 | 정적 (월/분기) | **실시간 (AI+제보)** |
| **데이터 범위** | 상업용만 | 주요 시설 | **비공식 지리정보 포함** |
| **위험 분석** | 없음 | 과거 데이터 | **예측적·동적 스코어링** |
| **커뮤니티** | 일방향 | 일방향 | **양방향** |
| **오프라인** | 제한적 | 미지원 | **지원** |

**핵심**: VeriSafe는 "가장 빠른 길"이 아닌 **"가장 안전한 길"** 제공

---

## 📊 현재 VeriSafe 구현 상태

### ✅ 이미 구현된 기능
- ✅ 안전/빠른/대안 경로 3가지
- ✅ 위험 정보 실시간 표시
- ✅ 사용자 제보 시스템 (4단계 위저드)
- ✅ 외부 데이터 수집 (ACLED, GDACS, ReliefWeb)
- ✅ AI 위성 분석 (Sentinel Hub)
- ✅ 배지 시스템 (신뢰도)
- ✅ 즐겨찾기 및 최근 경로
- ✅ GPS 위치 추적

## 📝 세부 기능 명세

### 1. **긴급 SOS 버튼** ⭐⭐⭐

**위치**: 지도 화면 우측 하단 (빨간색)

**동작**:
1. 버튼 3초 길게 누르면 활성화 (오작동 방지)
2. GPS 좌표 + 시간 + 사용자 정보를 서버에 전송
3. 등록된 비상 연락망에 SMS/푸시 알림
4. 가장 가까운 안전 거점 (대사관, UN, 병원) 표시
5. 긴급 전화번호 바로 연결

**UI**:
```
┌─────────────────┐
│                 │
│      지도        │
│                 │
│          [🧭]  │
│          [📍]  │
│          [🆘]  │ ← 빨간색, 크게
└─────────────────┘
```

**코드 위치**: `EmergencyButton.js` (이미 있음) → 기능 강화

---

### 2. **위험 진입 경고** ⭐⭐⭐

**트리거**:
- 사용자가 위험 지역 1km 이내 진입 시
- 위험도 70점 이상 지역

**알림 방식**:
1. 전체 화면 경고 팝업
2. 진동 (3번)
3. 경고음 (선택 가능)
4. 음성: "경고! 위험 지역에 진입하고 있습니다"

**UI**:
```
┌─────────────────────────────┐
│  ⚠️ 위험 지역 진입 경고       │
│                             │
│  무력충돌 발생 지역          │
│  위험도: 85점               │
│                             │
│  [대안 경로 보기]  [계속]   │
└─────────────────────────────┘
```

---

### 3. **안전 체크인** ⭐⭐⭐

**동작**:
1. 사용자가 목적지 설정 시 자동으로 체크인 알림 등록
2. 도착 예정 시간 + 30분 후 자동 체크인 요청 알림
3. 미체크인 시 비상 연락망에 자동 알림
4. 수동 체크인도 가능

**UI** (프로필 탭):
```
┌──────────────────────────┐
│  안전 체크인              │
│                          │
│  [✓] 자동 체크인 활성화  │
│                          │
│  비상 연락망:            │
│  • 김철수 (+211-xxx)     │
│  • UN 안전담당자         │
│                          │
│  [연락망 추가]           │
└──────────────────────────┘
```

---

### 4. **안전 거점 표시** ⭐⭐

**카테고리**:
- 🏛️ 대사관/영사관
- 🏥 병원 (국제 수준)
- 🛡️ UN 시설
- 🚓 경찰서 (안전)
- 🏨 안전 호텔
- ⛪ 교회/사원 (임시 대피소)

**지도 표시**:
- 초록색 마커
- 탭하면 상세 정보 + 경로 찾기
- 필터로 카테고리별 표시/숨김

**데이터 소스**:
- OpenStreetMap
- HDX (Humanitarian Data Exchange)
- 사용자 제보

---

### 5. **도착 예정 시간 (ETA)** ⭐⭐

**표시 정보**:
- 도착 예정 시간
- 남은 거리
- 평균 위험도
- 예상 소요 시간

**UI** (경로 화면 상단):
```
┌─────────────────────────────┐
│  🕐 15:30 도착 예정          │
│  📏 12.5 km | ⏱️ 25분       │
│  🛡️ 안전도: 78/100 (양호)   │
└─────────────────────────────┘
```

### 6. **음성 위험 경고** ⭐⭐

**시나리오**:
```
[운전 중]
→ "500m 앞에 검문소가 보고되었습니다"
→ "2km 앞에 시위가 진행 중입니다. 우회 경로를 제안합니다"
→ "경고! 위험 지역에 진입하고 있습니다"
```

**기술**:
- React Native TTS (Text-to-Speech)
- 한국어, 영어, 프랑스어, 스페인어 지원
- 배경음악 자동 중지

### 7. **비상 연락망** ⭐⭐⭐

2. 실시간 위치 공유 on/off
3. SOS 발동 시 자동 SMS + 푸시
4. 안전 체크인 미확인 시 알림

**설정 화면**:
```

│     [위치 공유 중] [편집]   │
│                              │
│  2. 박영희 (가족)            │
│     +82-10-XXXX-XXXX        │
│     [위치 공유 OFF] [편집]  │
│                              │
│  [+ 연락처 추가]             │
```

**상용 네비게이션**:
- 목적: 빠르고 편리하게
- 사용자: 일반 시민
- 환경: 안정적인 도시

**VeriSafe**:
- 목적: **안전하게 생존**
- 사용자: **분쟁 지역 활동가**
- 환경: **위험한 분쟁취약국**

### **핵심 기능 (상용 네비게이션에 없는 것)**

1. ⭐⭐⭐ **긴급 SOS** - 생명 직결
2. ⭐⭐⭐ **위험 진입 경고** - 예방 중심
3. ⭐⭐⭐ **안전 체크인** - 생존 확인
4. ⭐⭐⭐ **비상 연락망** - 실시간 위치 공유
5. ⭐⭐ **안전 거점 표시** - 대사관, UN, 병원

**이 기능들은 Google Maps, Waze에도 없습니다!**
