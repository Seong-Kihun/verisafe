# VeriSafe 디자인 양식

## 주요 기능 및 UI/UX 설계

### 1. 지도 탭 (Map Tab)
**핵심 기능 요소**

```
┌─────────────────────────────────────────┐
│  [검색바] 🔍 위치 검색                    │
├─────────────────────────────────────────┤
│                                         │
│    ┌─────────────────────────────┐     │
│    │     실시간 지도 뷰           │     │
│    │  • 위험 마커 (색상별 표시)   │     │
│    │  • 안전 쉼터 (파란색)        │     │
│    │  • 경로 폴리라인 (다중)      │     │
│    │  • 사용자 위치 (GPS)        │     │
│    └─────────────────────────────┘     │
│                                         │
│  [위험 필터 칩]                          │
│  [무력충돌] [시위] [검문소] [도로손상]    │
│                                         │
│  [레이어 토글 버튼] 📍                   │
│  [경로 찾기 FAB] ➡️                     │
└─────────────────────────────────────────┘
```

**직관적 UI/UX 요소**
- **색상 코딩**: 위험도별 색상 (빨강=높음, 주황=중간, 노랑=낮음)
- **실시간 GPS**: 사용자 현재 위치 자동 추적
- **필터링**: 위험 유형별 토글 필터 (다중 선택 가능)
- **레이어**: 위험/안전쉼터/경로 레이어 ON/OFF
- **인터랙션**: 마커 탭 → 상세 정보 바텀시트

**기능 연결**
- 검색바 → SearchScreen (자동완성)
- 마커 탭 → PlaceDetailSheet (상세 정보)
- FAB 버튼 → RoutePlanningScreen
- 필터 칩 → 지도 마커 필터링
- 레이어 버튼 → LayerToggleMenu


### 2. 경로 찾기 (Route Planning)
**핵심 기능 요소**

```
┌─────────────────────────────────────────┐
│  ← 경로 찾기                             │
├─────────────────────────────────────────┤
│  🔵 출발지                               │
│  [현재 위치 사용] [지도에서 선택]         │
│                                         │
│  🔴 목적지                               │
│  [최근 목적지] [즐겨찾기] [지도 선택]     │
│                                         │
│  🚶 이동수단                             │
│  [도보] [자전거] [자동차] [대중교통]      │
├─────────────────────────────────────────┤
│  💡 10개 경로 옵션                       │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 경로 #1 [추천] 안전 우선           │   │
│  │ 🕐 15분  📏 2.3km  ⚠️ 위험도 2    │   │
│  │ ────────────────────────         │   │
│  └─────────────────────────────────┘   │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 경로 #2 빠른 경로                 │   │
│  │ 🕐 12분  📏 1.8km  ⚠️ 위험도 5    │   │
│  │ ────────────────────────         │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [정렬: 시간순 ▼]                       │
└─────────────────────────────────────────┘
```

**직관적 UI/UX 요소**
- **빠른 입력**: 현재 위치, 최근 목적지, 즐겨찾기 원터치
- **다양한 옵션**: A* 알고리즘으로 10개 대안 경로
- **비교 지표**: 시간/거리/위험도 한눈에 비교
- **정렬**: 시간/거리/위험도 기준 정렬
- **시각적 구분**: 추천 경로 배지, 색상 코딩

**기능 연결**
- 경로 선택 → RouteHazardBriefing (위험 브리핑)
- 브리핑 확인 → MapScreen (경로 표시)
- 출발지/목적지 버튼 → SearchScreen
- 이동수단 변경 → 경로 재계산

### 플로우 1: 안전한 경로 찾기
**사용자 여정 단계별 화면 흐름**

```
[시작]
  ↓
[MapScreen 진입]
  • 현재 위치 자동 표시
  • 주변 위험 마커 확인
  ↓
[의사결정 지점 1] 어디로 가고 싶은가?
  ↓
[FAB 버튼 탭] "경로 찾기"
  ↓
[RoutePlanningScreen]
  • 출발지: 현재 위치 (자동)
  • 목적지: 입력 필요
  ↓
[인터랙션 1] 목적지 입력
  ├─ 옵션 A: 검색 (자동완성)
  ├─ 옵션 B: 최근 목적지 선택
  ├─ 옵션 C: 즐겨찾기 선택
  └─ 옵션 D: 지도에서 선택
```
